Game-X Level Editor

C++   OpenGL3.3   SDL2.0   CEGUI

User-Friendly Level Editor that helps to create 3D Environments.

  1. Stable For OBJ models [Supports other formats]
  2. Config. file that enables flexibility for the user.
  3. General GLSL inputs that can be used for any user made GLSL files.
  4. -Navigation mesh That helps to find walk-able areas of the map.
  5. -Relatively good graphics with Directional & Point Light Support along with Normal mapping in Tangent Space,Reflective water surface and Shadows.

SIC/XE Assembler

Java   JavaFX

Two Pass Assembler that translate SIC/XE to obj code.

  1. Using the user-friendly GUI you can load your code file or you can write it directly into the Editor.
  2. There are 2 modes which you can write code in : Restricted and Free Format Modes.
  3. Restricted which have certain positions for instruction format specified in the project PDF.
  4. After Successful Assemble the user can check the output from the 'Assemble' menu like the ListFile and the ObjFile.
